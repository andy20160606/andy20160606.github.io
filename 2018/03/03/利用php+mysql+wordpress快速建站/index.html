杂谈

　   快速建立一个网站对于大部分人也许并不是一件容易的事，尤其是建立一个相对美观的网站。wordpress 可以说是基于php的开源的网站模版吧，用起来可以说相当简单，虽然模版用了php语言，但是作为使用者来建立网站，一点也不需要用到关于php语言方面的东西。还有就是wordpress建立网站的效率相当高，国内很多云主机商都提供了相关服务。

关于建立的网站可定制性

     可定制性相当高，按照步骤建立完成后（建立基本就是鼠标点点点），可以通过web可视化窗口进行更改相应设置，包括换主题，更改标题，发表文章，建立页面，还有一些非常好用的插件。如果要想用来做电商网站也是可以的（需要用一些其他插件，没有深入研究）。

开始前准备

1，php环境
2，mysql 数据库服务器
3，wordpress 的模板包
4， php开发工具（可选，可以通过控制台直接运行，开发推荐使用phpstorm）

php环境配置

1.在 php官网下载php环境的zip包，解压到D盘（linux环境自行安装）
必要配置：
zip版本的php环境需要配置一下根目录下的php.ini(可以通过php.ini-development建立副本) 打开php.ini,将相关的扩展打开

extension=mysqli 前面的 ; 删掉，
extension=curl  前面的 ; 删掉，
extension_dir = "ext" 前面的; 删掉
  1. 用phpmyadmin作为php环境（个人没有用过，应该要比zip包版本的php环境好用） （如果选用1，那么2可省略了，用2则1可省略）

mysql环境配置

在 mysql官网下载相应的mysql数据库，并且配置好用户名密码，建立数据库 wordpress

wordpress 模板包下载

在 wordpress中文官网 把zip包下载下来并且解压缩

运行

window环境下使用cmd命令（hp.exe 是php环境的位置， wordpress 是 wordpress模板包的位置）

D:\php_7.2.3_home\php.exe -S localhost:80 -t D:\wordpress

运行成功截图：

最后通过浏览器访问 localhost 进行相应设置，填写上数据库用户名密码，就搞定了

可能遇到的问题

  wordpress是非常成熟的，我相信大部分问题是php环境的问题，如果上面我配置的php环境不通，请尝试别的方法

结语

希望大家建站愉快，有什么问题欢迎发送邮件给我

引子

如果您在Ubuntu中忘记了用户密码，请不要担心。您可以很容易地重置Ubuntu密码。本指南展示了一种在Ubuntu中更改root密码的简
单方法。该方法也应该适用于其他Linux发行版

step1 : 从恢复模式重置Ubuntu密码

打开电脑。转到Grub菜单。一般来说，它会自动出现，如果没有按住shift键，直到出现启动菜单。

如果您正在使用Oracle VirtualBox或VMWare，则必须在出现Oracle或VMWare徽标时按住Shift键。

在grub菜单中，选择“Ubuntu的高级选项”：

在这里，你会看到选择进入恢复模式：

它会带给你一个黑屏，有几行输出显示在闪存中。等待几秒钟

step2 ：放到root shell提示符下

现在你将出现不同的恢复模式选项。这里你需要选择“ Root Drop into root shell prompt ”。只需按回车键选择此选项。如下图所示：

step3 ：用写入权限重新安装根目录

您需要具有对根分区的写入权限。默认情况下，它只有只读权限。

使用下面的命令来重写它的写权限：

mount -rw -o remount /

step4 ：重置用户名或密码

在这里，你将会看到root权限。使用以下命令列出所有可用的用户：

ls /home

从前面的命令中选择您要重置或（说）破解密码的“ 用户名 ”。现在，使用以下命令重置所选“ 用户名 ” 的密码：

passwd username

它会提示输入新的密码。输入新密码两次：

Enter new UNIX password:
Retype new UNIX password:

step 5 ： 退出 root shell 提示符

您刚刚成功重设密码。现在退出root shell提示符：

exit

当您退出时，您将回到恢复模式菜单。在这里选择正常启动选项。

总结

其他linux发行版大多可以使用此方法，如果行不通请使用专用的方法
